## Training

### Quick start (recommended)

```bash
bash train.sh
```

This single command runs the full pipeline:
1. **Preprocess** вЂ” downloads BANKING77 CSVs from GitHub, selects top-30 intents, builds ChatML prompts, stratified train/val/test split
2. **Fine-tune** вЂ” trains Llama 3.1-8B-Instruct with QLoRA + EarlyStopping
3. **Evaluate** вЂ” prints accuracy + classification report on the test set
4. **Save** вЂ” saves checkpoint to `model_checkpoint/`

### Resume from checkpoint

```bash
bash train.sh --resume
```

### Run steps individually

```bash
# Step 1: Preprocess only
python scripts/preprocess_data.py --config configs/train.yaml

# Step 2: Train only
python scripts/train.py --config configs/train.yaml

# Step 3: Resume from specific checkpoint
python scripts/train.py --config configs/train.yaml --resume --checkpoint_dir outputs/checkpoint-276
```

## Hyperparameters

| Parameter | Value | Notes |
|---|---|---|
| Base model | `Meta-Llama-3.1-8B-Instruct-bnb-4bit` | 4-bit quantized via Unsloth |
| LoRA rank (r) | 32 | Higher rank = more capacity |
| LoRA alpha | 32 | Equal to r (standard practice) |
| LoRA dropout | 0 | Unsloth-optimized |
| Target modules | q/k/v/o/gate/up/down proj | All projection layers |
| Learning rate | 5e-5 | Cosine decay |
| Batch size (per device) | 4 | |
| Gradient accumulation | 4 | Effective batch = 16 |
| Epochs | 8 (max) | EarlyStopping patience=2 |
| Max sequence length | 512 | Accommodates ChatML + intent list |
| Optimizer | AdamW 8-bit | Memory-efficient |
| Warmup ratio | 5% | |

## Inference

### Via shell script

```bash

# Custom message
bash inference.sh "I lost my credit card"
```

### Via Python

```python
from scripts.inference import IntentClassification

# __init__: loads config, tokenizer, model checkpoint
clf = IntentClassification("configs/inference.yaml")

# __call__: returns predicted intent label
label = clf("I lost my card and need a replacement")
print(label)  # в†’ "card_arrival"

label = clf("What is the exchange rate today?")
print(label)  # в†’ "exchange_rate"
```
